# 免费A股日线数据接口

## 声明

* 这是一个免费的A股日线数据接口（盘后数据）。可供量化学习使用。
* 包含Java与Python两个版本。欢迎贡献更多语言的实现。
* 仅供学习交流使用，严禁用于任何商业用途。
* 不保证数据的更新实时性和准确性，使用本接口产生的一切后果自负。

## 数据更新

* 正常情况下，数据在每个交易日的 `15:10` 左右更新。
* 如果出现意外，更新时间可能推迟到 `15:30` 或 `16:00`。
* 如果 `16:00` 后数据仍未更新，则说明当天的更新可能出现了问题。

## 数据内容

* 接口提供约 3000 支股票的数据。
* 包含过去 30 个交易日以及当日的日线行情（开盘价、收盘价、最高价、最低价、成交量、成交额）。
* 只包含在过去 30 个交易日内均有交易数据的股票，因此不包含近期上市的新股。
* 一般来说可以在 5 秒内获取并解析全部数据，只需要传输2MB左右的数据，远小/快于其他库实现。
